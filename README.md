💰 Saving Calculator

A simple and interactive Saving Calculator built using HTML, CSS, and JavaScript. This project helps users estimate their future savings based on monthly deposits, interest rate, and saving duration.

🚀 Features

- Calculate future savings amount
- Monthly savings input
- Annual interest rate input
- Saving period in years
- Instant calculation results
- User-friendly interface
- Responsive design

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📋 How It Works

1. Enter your monthly savings amount.
2. Enter the annual interest rate (%).
3. Enter the saving period (years).
4. Click the Calculate button.
5. View your estimated future savings amount instantly.

🎯 Purpose

This project was created to practice JavaScript calculations, form handling, and responsive web design while building a useful financial tool.
